ALOS - JAXA EORC Global Digital Surface Model (DSM)
Version 3.2/3.1

DSM - Height above sea level (avg from AW3D)
MSK - Mask information (Cloud, snow, sea, DEM for complement processing)
STK - Stack number of DSM scenes used to produce DSM
QAI - Quality Assurance Information
HDR - Metadata
LST - List of Scene IDs used to produce DSM

Resolution: 1 arcsecond (~30m), latitude dependent
File resolution: 1x1 degree Latitude/Longitude tile

see aw3d30v3.2 product specification PDF for more information about structure
